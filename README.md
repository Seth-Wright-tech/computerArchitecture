Three projects from my computer architecture class.

Project 1 : 
cache simulator that simulated different levels of a cache system with a few different replacement protocols.

Project 2 : 
Simulation of an out-of-order processor using Tomasulo's algorithm along with branch prediction.

Project 3 :
Simulates different protocols for cache coherency in an out-of-order processor, including MOSI, MESI, and MOESIF.
